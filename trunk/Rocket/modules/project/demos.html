Prepackaged demos

All demos can be found under the www directory within the rocket download. The java source for each widget exists in a parallel test package for the said component.

Demo Server required Info Demo class
DragNDrop No Drag n drop in action rocket.dragndrop.test.client.DragNDropTest
Comet Yes - run comet.launch Cometting of java objects to the client from a server rocket.remoting.test.comet.client.CometTest
Logging No A logging framework with support for named loggers with different levels, and the possibility to select (via a query parameter) whether the generated javascript contains any logging statements. rocket.logging.test.application.client.LoggingApplication
Selection No Programmatic interaction with selections. rocket.selection.test.selection.client.SelectionTest
StyleSheet No Interactive test that demonstrates programmatic manipulation of stylesheets, rules and css properties rocket.style.test.stylesheet.client.StyleSheetTest
InteractiveList No Tests a test facility for any List rocket.testing.test.interactivelist.client.InteractiveListTest
InteractivePanel No Tests a test facility for any Panel rocket.testing.test.interactivepanel.client.InteractivePanelTest
WebPageTestRunnerTest No Tests a facility for writing interactive functional tests of widgets and other components which require a user to confirm something worked. rocket.testing.test.webpagetestrunner.client.WebPageTestRunnerTest
StackTrace No Demonstrates stacktraces in web mode (outside hosted mode). rocket.util.test.stacktrace.client.StackTraceTest
AccordionPanel No Accordion widget rocket.widget.test.accordionpanel.client.AccordionPanelTest
AutoCompleteTextBox No AutoCompleteTextBox widget in action, matches againsta prepared list(shown on page). rocket.widget.test.autocompletetextbox.client.AutoCompleteTextBoxTest
BasicWidgets yes - run basicWidgets.launch to test FormPanel All GWT basic widgets rewritten to be more easily subclasses, accept hijacked elements from the dom and more, whilst maintaining the original functionality. rocket.widget.test.basicwidgets.client.BasicWidgetsTest
DivPanel No A panel that uses a DIV panel as the root, with A DIV surrounding each child widget. rocket.widget.test.divpanel.client.DivPanelTest
HyperLinkPanel No A panel that uses a A panel as the root with the ability to add child widgets. rocket.widget.test.hyperlinkpanel.client.HyperlinkPanelTest
ImageFactory No Demonstrates ImageFactory in action. Try with different browsers and observe appropriate usage of data urls. rocket.widget.test.imagefactory.client.ImageFactoryDemo
Menu No A rich the menu system that supports control over the way menu lists open and a context menu.
  • Down - like traditional menus.
  • Up - like the windows start menu
  • Right - most sub menus typically open to the right
  • Left - when most sub menus are placed on the rigth edge of the screen they open to the left.
  • ContextMenu - a panel that includes a ContextMenu which is triggered by right mouse clicks
 rocket.widget.test.menu.client.MenuTest
OrderedListPanel No A panel that uses a OL panel as the root with a LI surrounding each child widget. rocket.widget.test.orderedlistpanel.client.OrderedListPanelTest
ResizablePanel No A panel that can be used to resize a child widget by dragging the edges. rocket.widget.test.resizablepanel.client.ResizablePanelTest
Slider No Demonstrates several slider widgets (with different orientations). rocket.widget.test.slider.client.SliderTest
SortableTable No A powerful fast sort table implementation. rocket.widget.test.sortabletable.client.SortableTableTest
SpanPanel No A panel that uses a SPAN panel as the root with SPAN surrounding each child widget. rocket.widget.test.spanpanel.client.SpanPanelTest
SplitterPanel No Demonstrates several splitter panel widgets (unlike the GWT splitter panels both panels accept more than just 2 child widgets). rocket.widget.test.splitterpanel.client.SplitterPanelTest
TabPanel No A rich tab panel implementation with different flavours that allow customisation of where the tabs appear (above, below, to the right or left) of the content. rocket.widget.test.tabpanel.client.TabPanelTest
UnorderedListPanel No A panel that uses a UL panel as the root with a LI surrounding each child widget. rocket.widget.test.unorderedlistpanel.client.UnorderedListPanelTest
Viewport Yes (run viewport.launch) A simple yet powerful tile based viewport widget visually and functionally similar to the Google map map widget. rocket.widget.test.viewport.client.ViewportTest
ZebraFlexTable No A flextable that automatically adds an even or odd style to even or odd rows. rocket.widget.test.zebraflextable.client.ZebraFlexTableTest

ImageFactory test

  • Lists several images of varying size, which should result in some being loaded from the server and others using data urls
  • Depending on the browser the generator will create different urls for each of the images, mostly due to a browsers limits or lack of data url support.
  • For InternetExplorer all images will be sourced from the server.
  • For FireFox and Safari all images will contain data urls.
  • For Opera the first two images (the cog and tomato) will use data urls, whilst the largest image will be loaded from the server because its data url form exceeds the supported browser size of 4K(kiwi)

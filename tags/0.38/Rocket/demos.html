Prepackaged demos

All demos can be found under the www directory within the rocket download. The java source for each widget exists in a parallel test package for the said component.

SelectionNoProgrammatic interaction with selections.
StyleSheetNoInteractive test that demonstrates programmatic manipulation of stylesheets, rules and css properties
InteractiveListNoTests a test facility for any List
InteractivePanelNoTests a test facility for any Panel
WebPageTestRunnerTestNoTests a facility for writing interactive functional tests of widgets and other components which require a user to confirm something worked.
StackTraceNoDemonstrates stacktraces in web mode (outside hosted mode).
AccordionPanelNoAccordion widget
AutoCompleteTextBoxNoAutoCompleteTextBox widget in action, matches againsta prepared list(shown on page).
BasicWidgetsNoAll GWT basic widgets rewritten to be more easily subclasses, accept hijacked elements from the dom and more, whilst maintaining the original functionality.
BreadcrumbPanelNoA simple breadcrumb panel that can pop and push crumbs
DivPanelNoA panel that uses a DIV panel as the root, with A DIV surrounding each child widget.
GridViewNoA grid base panel that lazily initiates each cell as well as supporting a window of a much larger dataset.
HyperLinkPanelNoA panel that uses a A panel as the root with the ability to add child widgets.
MenuNoA rich the menu system that supports control over the way menu lists open and a context menu.
  • Down - like traditional menus.
  • Up - like the windows start menu
  • Right - most sub menus typically open to the right
  • Left - when most sub menus are placed on the rigth edge of the screen they open to the left.
OrderedListPanelNoA panel that uses a OL panel as the root with a LI surrounding each child widget.
PagerNoA widget that can be used to create a paging navigation control.
ResizablePanelNoA panel that can be used to resize a child widget by dragging the edges.
SliderNoDemonstrates several slider widgets (with different orientations).
SortableTableNoA powerful fast sort table implementation.
SpanPanelNoA panel that uses a SPAN panel as the root with SPAN surrounding each child widget.
SplitterPanelNoDemonstrates several splitter panel widgets (unlike the GWT splitter panels both panels accept more than just 2 child widgets).
StyleSheetPickerNoA simple widget that provides the capability to allow the user to select any prepared stylesheet (all are disabled accept for the current).
TabPanelNoA rich tab panel implementation with different flavours that allow customisation of where the tabs appear (above, below, to the right or left) of the content.
TreeNoA rich tree widget that supports adding widgets to each individual node.
UnorderedListPanelNoA panel that uses a UL panel as the root with a LI surrounding each child widget.
ViewportYes (run viewport.launch)A simple yet powerful tile based viewport widget visually and functionally similar to the Google map map widget.
ZebraFlexTableNoA flextable that automatically adds an odd or even style to odd or even rows.

Things to test

The server contains a servlet that returns a tile (sub image) of a much larger world map, in a similar manner to that found in googlemaps albeit in a much simpler manner. The zooming ability is achieved by simply scaling this same image when creating a tile.

  • The user may scroll around the world map by dragging.
  • If the user scrolls too much too the left, right, top or bottom the map wraps round.
  • The user may change the zoom setting by entering a new integer value and hitting enter in the zoom textbox.
  • The dotted/dashed lines on the map are part of the image and not the actual markers for the tile grid.
  • A embedded tomcat is required because of the need for a custom servlet which serves the individual tiles

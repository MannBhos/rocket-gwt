A Slider is a widget which uses a sliding handle to set a value. It comes in two forms a HorizontalSlider and a VerticalSlider.
Things to test
  • Entering number values in any of the TextBoxes present will update the so named property of both Sliders.
  • Dragging the handle will update the appropriate value label(This is accomplished by a attaching a listener to each Slider).
  • After letting go of the handle it should still be possible to drag/select text through out the page - When dragging begins the selection behaviour of the browser is disabled. It should be reenabled when the drag cancelled is let go.
  • Click on the background area of either slider and verify that the handle moves towards jumping by the bigDelta amount.
  • It is not necessary to drag the handle along the slider. It is possible to drag outside this area and the handle will continue to move accordingly along the slider's axis.
  • Dont forget to resize the window so that scrollbars appear and scroll the viewport and run tests again.

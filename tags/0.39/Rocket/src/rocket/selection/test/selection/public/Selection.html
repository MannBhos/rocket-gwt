This page includes a button which runs a variety of selection tests with some input required from the user. Directions are given as appropriate.

Things being tested
  • Clearing any text selection
  • Text selection disabling
  • Text selection enabling
  • Testing if a text selection is present
  • Programmatically setting a selection
  • Programmatically reading the selection range
  • Programmatically deleting the selection range

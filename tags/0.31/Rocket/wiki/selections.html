Purpose

The purpose of the rocket.selection package is to expose in a browser independent manner the ability to work with selections. Currently there are two main very different mechanisms to work with selections, the Internet Explorer way and the standards way.

Features

Using rocket.selection.client.SelectionHelper and accompanying classes it is possible to perform the following actions.

  • Test if a selection exists
  • Delete from the DOM a selection
  • Read the start and end points of a selection.
  • Programmatically set the start and end points of a selection
  • Surround a selection with an element.
  • Extract the DOM nodes within a selection and make them a child of another element.
